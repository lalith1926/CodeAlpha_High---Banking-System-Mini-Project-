 Description

This project is a simple Banking System application developed as part of the CodeAlpha internship program. It simulates basic banking operations such as account creation, deposits, withdrawals, and balance checking.

The goal of this project is to demonstrate fundamental programming concepts like object-oriented design, file handling (if used), and user interaction.

 Features

Create a new bank account

Deposit money

Withdraw money

Check account balance

Display account details

Simple and user-friendly interface
